# Kubernetes & DevOps Journey 🚀

This repository contains the projects, configurations, and experiments I've built while learning Kubernetes, Docker, Linux, and DevOps.

Instead of following only tutorials, I use this repository as a hands-on lab where I deploy applications, manage infrastructure, and explore real-world DevOps concepts.

### MySQL

* StatefulSet
* ConfigMaps

### Nginx

* Nginx container deployments

### Django Notes App

A containerized Django application deployed and managed on Kubernetes.

## Getting Started

Clone the repository:

```bash
git clone git@github.com:harsh1922/Kubernetes.git
cd Kubernetes
```

Apply resources:

```bash
kubectl apply -f apache/
kubectl apply -f mysql/
```
